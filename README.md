
## Description

Enables to create one time link. 

## Installation

1. Creates .env file
```bash
$ cp .env.example .env
```
2. Dependencies install
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
